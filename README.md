
Invasión Espacial
¡Bienvenido al juego Invasión Espacial! Derrota a los invasores espaciales y acumula puntos para demostrar tus habilidades. ¡Prepárate para la acción intergaláctica!

Instrucciones de juego
Utiliza las teclas izquierda y derecha para mover tu nave espacial.
Presiona la tecla espacio para disparar a los invasores.
¡Evita colisionar con los invasores y destrúyelos para ganar puntos!
Controles
Tecla Izquierda: Mover a la izquierda
Tecla Derecha: Mover a la derecha
Tecla Espacio: Disparar
Puntaje
Tu puntaje se muestra en la esquina superior izquierda. ¡Cuantos más invasores derrotes, mayor será tu puntuación!
